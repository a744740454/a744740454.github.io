docker 容器互联

当你创建了多个容器后，你可以直接使用ip地址ping通，但是我们需要考虑一个情况

某个容器修改了ip,这样子的话其余连接该容器的容器都需要更换成新的ip地址。是不是很麻烦

这种时候我们需要一种技术，通过连接主机名，也就是容器名，我们就可以让容器之间互相连接。
